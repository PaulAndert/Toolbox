Konventionen für den Python-Code:

Variablen sollten in den gewünschten Type gecastet werden.

Error ist standardmäßig auf False, wenn auf True gesetzt wird, kann die eigene Errormeldung ausgegeben werden.

Nach diesem Schema sollte alles funktionieren:

a = int(a)
b = int(b)
c = float(c)

ausgabe_2 = b - c
a = ausgabe_1
b = ausgabe_2
