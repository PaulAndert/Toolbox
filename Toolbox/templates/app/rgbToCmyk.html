Tutorial zum Selbermachen:

Wenn man R: 0, G: 0 und B: 0 hat ist es einfach C: 0, M: 0, Y: 0 und K: 1

wenn das nicht der fall ist rechnet man zuerst
gelb = 1 - blau / 255
cyan = 1 - rot / 255
magenta = 1 - grün / 255

Dann muss man den kleinsten der drei neu berechneten werte finden, dieser ist CMY_minimum, damit kann man dann die entgültigen werte ausrechnen
cyan = (cyan - CMY_minimum) / (1 - CMY_minimum)
magenta = (magenta - CMY_minimum) / (1 - CMY_minimum)
gelb = (gelb - CMY_minimum) / (1 - CMY_minimum)
kontrolle = CMY_minimum
